# Краткое описание для резюме

## Вариант для раздела "Проекты"

**Linux Administration & Networking Practice**

Практический проект на Ubuntu Server 24.04 LTS в VirtualBox. Развернула и настроила несколько Linux VM без графического интерфейса, выполнила базовое администрирование и построила учебную сетевую инфраструктуру с маршрутизацией, firewall, DHCP и NAT.

Что делала:

- настраивала пользователей, группы, sudo-доступ, hostname, часовой пояс и синхронизацию времени;
- переводила сетевую конфигурацию с DHCP на статические параметры через netplan;
- устанавливала и настраивала OpenSSH Server, меняла порт, проверяла процессы и TCP-порты;
- работала с systemd-службами, cron, логами авторизации и системными журналами;
- анализировала процессы, CPU, RAM, swap, диски и файловые системы через top/htop/fdisk/df/du/ncdu;
- рассчитывала IPv4-сети, маски, диапазоны хостов, private/public-адреса;
- настраивала статическую маршрутизацию между VM и постоянные маршруты через netplan;
- измеряла пропускную способность сети через iperf3;
- писала и проверяла правила iptables, диагностировала доступность через ping и nmap;
- строила сеть из рабочих станций и Linux-маршрутизаторов, включала IP forwarding;
- анализировала маршрут пакетов через traceroute/tracepath и tcpdump;
- настраивала DHCP-сервер, fixed-address по MAC и обновление DHCP-аренды;
- настраивала NAT/SNAT/DNAT и проверяла TCP-доступ через telnet и Apache2.

## Короткий вариант для CV

Практический опыт администрирования Ubuntu Server: пользователи и sudo, SSH, systemd, cron, логи, мониторинг ресурсов, netplan, статическая маршрутизация, iptables, DHCP, NAT, tcpdump/traceroute-диагностика, iperf3/nmap/ping/telnet-проверки на нескольких виртуальных машинах.

## Более сильная формулировка для отклика

Получила практический опыт Linux и networking на нескольких Ubuntu Server VM: настраивала серверную ОС без GUI, пользователей и права, SSH, сетевые интерфейсы, systemd-службы, cron и логи; строила сетевую схему с маршрутизаторами, статическими маршрутами, IP forwarding, firewall на iptables, DHCP и NAT; проверяла результат через ping, ip r, netstat, nmap, iperf3, tcpdump, traceroute и telnet.

## Навыки

Linux, Ubuntu Server, Bash/CLI, systemd, SSH, sudo, cron, logs, netplan, IPv4, routing, iptables, DHCP, NAT, tcpdump, traceroute, nmap, iperf3, VirtualBox.

## Как рассказать на собеседовании

Я выполняла два практических проекта по Linux. В первом проекте подняла Ubuntu Server без GUI и прошла базовое администрирование: пользователи, sudo, сеть, SSH, обновления, время, процессы, диски, логи и cron. Во втором проекте работала уже с несколькими виртуальными машинами: рассчитывала подсети, настраивала маршруты, firewall, DHCP, NAT и проверяла прохождение трафика. Для меня самым полезным было научиться подтверждать результат: смотреть маршруты, порты, процессы, логи и сетевые дампы, а не просто считать, что команда сработала.
